# 交付物 (Deliverables)

**状态**: 待补充 — 源文档未包含交付物章节。

## 建议交付物清单

| 交付物 | 格式 | 描述 |
|--------|------|------|
| RTL代码 | .v / .sv | SPI2AXI IP 完整RTL设计代码 |
| SDC约束文件 | .sdc | 时钟、I/O延迟、CDC约束 |
| 验证环境 | SystemVerilog | UVM或自检查验证环境 |
| 测试用例 | .sv | 功能测试和时序测试用例 |
| 设计文档 | .md / .html | HLD + LLD 设计规格书 |
| 时序报告 | .rpt | 综合后时序分析报告 |
| 综合脚本 | .tcl | Synopsys DC综合脚本 |
