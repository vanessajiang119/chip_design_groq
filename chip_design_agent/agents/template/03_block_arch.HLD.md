**主状态机状态迁移**:

```
                    ┌──────────────────────────┐
                    │                          │
                    ▼                          │
  [IDLE] ──(配置使能)──► [CFG_LOAD] ──► [RUNNING]
                    │                          │
                    │  (异常)                   │  (完成)
                    │                          │
                    ▼                          ▼
                 [ERROR] ──(软复位)──► [IDLE]  [DONE] ──(读状态)──► [IDLE]
```

| 状态 | 描述 | 入口动作 | 出口条件 | 对应 LLD 状态编码 |
|------|------|---------|---------|-----------------|
| IDLE | 空闲等待 | — | 软件使能且数据就绪 | `3'b000` |
| CFG_LOAD | 加载配置参数 | 锁存 CSR 到工作寄存器 | 配置加载完成 | `3'b001` |
| RUNNING | 数据流处理中 | 启动数据引擎 | 所有数据完成 / 出错 | `3'b010` |
| DONE | 处理完成 | 置位状态位、产生中断 | 软件读状态寄存器 | `3'b100` |
| ERROR | 错误状态 | 记录错误码、产生中断 | 软件写软复位 | `3'b101` |
