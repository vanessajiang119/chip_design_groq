## skill定义

将以下需求，转换为一个AI专用的skill：web_search_tavily
skill 路径在放在./agents/skills

## 基础信息

技能名称：实时联网网页搜索
技能唯一标识：web_search_Tavily
适用主体：所有事实类、时效类、外部知识类AI Agent
核心引擎策略：优先Tavily高精度AI检索，接口异常/结果过少自动降级通用网页搜索兜底

## 调用强制触发规则

1. 问题含最新时间、实时数据、行业资讯、政策变动、外网资料、未收录知识库内容，必须调用本搜索
2. 模型静态知识存在6个月以上滞后性内容，禁止自行作答，强制联网检索
3. 产品参数、赛事行情、学术资料、技术文档、海外信息统一走本技能
4. 简短常识、固定公理、内置知识库已有内容，禁止滥用搜索

## 禁止调用场景

1. 纯创意写作、情绪对话、主观感悟、无事实依据闲聊
2. 本地私有数据、内部涉密信息、不可公开网络内容

## 检索执行固定流程

1. 接收标准化搜索关键词，剔除口语化冗余，优化检索句式
2. 优先发起Tavily检索，开启自动重试机制
3. 连续重试失败/有效结果不足2条，自动切换通用网页搜索补全
4. 合并双引擎结果，完成URL去重、同源标题去重、低质站点过滤
5. 清洗网页冗余内容（广告、导航、页脚），提炼核心有效信息
6. 统一封装带来源链接、时效、权重的结构化检索素材
7. 回传给大模型进行整合作答

## 内置能力清单

- 错误重试：接口超时、限流、服务异常自动重试3次，间隔梯度递增
- 多层去重：URL精准去重 + 标题语义相似度去重
- 内容提纯：自动过滤营销软文、水帖、垃圾资讯
- 来源溯源：所有结果强制保留原始URL，作答必须标注引用
- 深度切换：简易快速检索 / 深度全面检索双模式
- 时效筛选：支持限定今日/本周/本月/全时段内容筛选

## 入参规范

1. search_query：必填，精简精准检索词，语句通顺无歧义
2. search_depth：可选，basic快速 / advanced深度，默认advanced
3. max_results：可选，返回结果数量，默认6条
4. time_range：可选，now/week/month/all，默认all
5. need_full_content：可选，是否抓取全文，默认false

## 模型作答强制约束

1. 严格依托搜索结果作答，严禁脱离素材编造事实
2. 多源信息冲突时，优先采信官方平台、权威媒体、头部行业站点
3. 所有数据、事件、观点必须标注对应来源链接
4. 检索内容不足时如实说明，不强行拼凑答案
5. 时效性内容优先选用近期检索结果，舍弃老旧过期信息

## 异常返回话术

1. 检索接口异常：当前网络检索服务临时异常，暂时无法获取外网实时信息
2. 无相关结果：暂未检索到公开权威相关资料，相关内容仅供参考
3. 额度耗尽：联网搜索次数已用尽，停止外部检索
