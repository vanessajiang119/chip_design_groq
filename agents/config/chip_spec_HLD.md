# Skill 定义

将以下需求，转换为一个专业芯片的skill：chip_spec_hld

# High-Level Design Spec (HLD，高层设计规格书)

HLD 是系统/模块的整体架构蓝图。它回答 “这个 IP 要做什么、怎么和外界交互、主要特性是什么、资源预算如何” 等宏观问题。典型内容：

- 功能概述与目标
- 外部接口定义（信号列表、时序协议）
- 顶层架构框图
- 数据流与控制流
- 主要特性、可配置参数
- 性能/功耗/面积目标
- 应用场景
- 假设与约束

# 相关skills：

chip_spec_lld，用于生成芯片设计的LLD；

# HLD和LLD在设计流程中如何配合使用？

1. **HLD 先行**：架构师 / 系统工程师完成 HLD → 团队 Review → 冻结外部接口与功能范围。
2. **LLD 基于 HLD 展开**：RTL 设计师 / 微架构工程师根据 HLD 写 LLD → Review → 冻结实现细节。
3. **RTL Coding**：严格按照 LLD 写代码，HLD 是验收标准。
4. **验证**：TB / 测试用例基于 HLD 写功能覆盖点，基于 LLD 写 corner case。
5. **后续变更**：任何改动必须同时更新 HLD + LLD，保持一致性。
