## 一、系统级功耗模型

### 1.1 TSP 芯片功耗

Groq TSP（第一代，14nm @ 900MHz）的功耗特征：

| 负载场景 | 功耗 | 注释 |
|---------|------|------|
| **空闲** | ~15W | 仅时钟维持 + SRAM 保持 |
| **典型推理负载** | ~40W | ML 推理的典型工作点 |
| **密集计算负载** | ~100-150W | MXM 满负荷运算 |
| **GroqCard 峰值** | 375W（卡级） | 含 SerDes + PCIe 接口 |
| **平均典型** | ~240W（卡级） | 含所有板级组件 |

### 1.2 功耗分解

```
TSP 芯片功耗分解（估算 @ 典型负载 ~40W）:
┌─────────────────────────┬──────┬──────────┐
│ 组件                    │ 功耗  │ 占比     │
├─────────────────────────┼──────┼──────────┤
│ MXM（矩阵计算阵列）      │ ~12W │ ~30%     │
│ MEM（SRAM 读写）         │ ~10W │ ~25%     │
│ VXM + SXM（向量/移位）   │ ~6W  │ ~15%     │
│ C2C SerDes（11 条链路）  │ ~5W  │ ~12.5%   │
│ ICU + 时钟分布           │ ~4W  │ ~10%     │
│ 其他（NoC, 控制逻辑等）  │ ~3W  │ ~7.5%    │
├─────────────────────────┼──────┼──────────┤
│ 合计                     │ ~40W │ 100%     │
└─────────────────────────┴──────┴──────────┘

C2C SerDes 功耗细节（估算）:
  每条链路 ~450mW（4 lanes × ~112mW/lane @ 30Gbps）
  11 links × 450mW ≈ 5W
```

### 1.3 GroqCard 板级功耗分解

```
GroqCard GC1-010B 功耗分解（典型 @ ~240W）:
┌─────────────────────────┬──────┬──────────┐
│ 组件                    │ 功耗  │ 占比     │
├─────────────────────────┼──────┼──────────┤
│ TSP 芯片                │ ~40W │ ~17%     │
│ PCIe Gen 4 ×16 接口     │ ~25W │ ~10%     │
```

| 层级 | 配置 | 组件 | 典型功耗 |

## 二、功耗密度分析

### 2.1 计算效率指标

| 指标 | Groq TSP | NVIDIA A100 | NVIDIA H100 |
|------|---------|------------|------------|
| **INT8 TOPS/W** | ~3.1 | ~1.56 | ~2.83 |
| **FP16 TFLOPS/W** | ~0.78 | ~0.31 | ~0.71 |
| **TOPS/mm²** | ~1.0 | ~0.27 | ~0.45 |
| **SRAM 能效** | 极高（无需 HBM） | 中（需 HBM 堆叠） | 中 |

注：Groq 在 14nm 工艺上达到的能效，同工艺下优于传统 GPU 架构。

### 2.2 每 TSP 的功耗效率优势分析

Groq TSP 的低功耗（~40W 典型）主要来自：
1. **无需外部 HBM/DDR** — SRAM 功耗远低于 HBM（~5 pJ/bit vs ~20 pJ/bit）
2. **无 cache coherency 开销** — 不需要探听过滤器等
3. **无动态调度硬件** — 无 out-of-order、无分支预测、无重排序缓冲
4. **确定性执行** — 无冒险检测/转发逻辑
5. **简化互联** — TSP 兼交换机，无需专用网络芯片

### 2.3 功率密度对比

```
功率密度对比（kW/机架）:
  GroqRack:    ~20 kW （72 TSPs, 标准风冷）
  NVIDIA DGX:  ~40 kW （8 H100s + 网络）
  HPC 典型:    ~30-50 kW/rack
  # Groq 在更低功率密度下提供高推理性能
  # 适合现有数据中心无需升级电力设施
```

## 三、散热方案

### 3.1 标准风冷设计

GroqRack 采用 **标准风冷**，不需要液冷：

```
GroqNode 4U 风冷设计:
┌──────────────────────────────────┐
│  Front (进气)                     │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐    │
│  │FAN1│ │FAN2│ │FAN3│ │FAN4│    │ → 气流
│  └────┘ └────┘ └────┘ └────┘    │
│  ┌──────────────────────────────┐│
│  │ GroqCard 1-8                 ││ → 热空气
│  │ (横置气流通道)               ││
│  └──────────────────────────────┘│
│  Back (排气)                      │
└──────────────────────────────────┘

每 GroqCard 的散热设计:
  ┌─────────────────────┐
  │ 散热器（fin stack）   │ ← 铝/铜鳍片
  │ ┌─────────────────┐ │
  │ │ TSP 芯片         │ │ ← 导热界面材料 (TIM)
  │ │ (Die)            │ │
  │ └─────────────────┘ │
  │ ┌─────────────────┐ │
  │ │ PCIe 连接器      │ │
  │ └─────────────────┘ │
  └─────────────────────┘
```

### 3.2 散热参数

| 参数 | 数值 |
|------|------|
| **风冷 TDP** | ~375W/卡（最大设计） |
| **气流方向** | 前→后 |
| **风扇类型** | N+1 冗余热插拔 |
| **进风温度** | 15-32°C（标准数据中心） |
| **出风温度** | 典型 < 65°C |
| **声学噪音** | 标准数据中心级别 |

### 3.3 微喷冷却研究（下一代）

**IEEE ITherm 2024 论文** 来自普渡大学 + Groq Inc. 描述了直接片上微喷冷却：

```
微喷冷却原理:
┌─────────────────────────────────┐
│  Coolant inlet                   │
│  ↓                                │
│  ┌─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┐  │
│  │○│○│○│○│○│○│○│○│○│○│○│○│○│  │ ← 微喷阵列
│  └─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┘  │    (喷嘴直径 ~100μm)
│  ┌─────────────────────────────┐│
│  │ TSP Die（热点区域）         ││ ← 直接冲击冷却
│  └─────────────────────────────┘│
│  Coolant outlet ↑                │
└─────────────────────────────────┘
```

| 参数 | 效果 |
|------|------|
| **喷嘴直径** | 定制化，匹配芯片热点 |
| **流量** | 根据功率分布调节 |
| **温度均匀性** | 芯片上温差 < 4°C |
| **热耦合减少** | LPU 与相邻 chiplet 间显著降低 |
| **应用场景** | 未来更高密度部署 |

### 3.4 散热路线

```
2020-2023: Gen 1 风冷
  ├── 标准数据中心兼容
  ├── ~40W/TSP 典型负载
  └── 无额外基础设施需求

2024-2025: Gen 2 增强风冷 + 微喷（研发中）
  ├── 兼容更高 TSP 功率
  ├── 定向热点冷却
  └── 更低结温 → 更低泄漏功耗

2026+: Gen 3 液冷/浸没（可能）
  ├── 更高密度部署
  ├── 更佳 PUE
  └── 匹配更大规模系统（> 10,440 TSPs）
```

## 七、关键结论

1. **极低功耗是架构优势** — ~40W/TSP 典型功耗使标准风冷即可满足，无需液冷
2. **能效领先同代产品** — 14nm 工艺下 INT8 TOPS/W 达到 ~3.1，优于 7nm A100
3. **微喷冷却储备先进** — 与普渡大学合作的定向热点冷却研究为更高密度做准备
4. **基础设施友好** — 20kW/rack 功率密度可部署在大部分现有数据中心
5. **电源架构符合标准** — 220VAC 输入 + 标准 PSU，无需特殊电力改造
